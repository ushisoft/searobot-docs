## 电池仓

![电池仓](/images-searobot/media/image5.png)

底盘电池支持快拆，按下电池仓盖板即可打开盖板。电池接口定义如下：

- 电池放电口：XT90 母头
- 电池充电口：XT60 母头
- BMS 接口：USB

::: warning 注意
更换电池后需重新接好三个插头，否则影响正常功能使用。
:::

## 电源盒

![电源盒](/images-searobot/media/image6.png)

### 48V 输入

带线引出，接头为XT60 公头，接入电源通讯面板中的电源输出口。

### 48V 接口定义

| 插头型号 | 引脚编号 | 引脚定义 |
|----------|----------|----------|
| 威浦WS20-6P | 1 | 正 |
| 输出参数 | 2 | 正 |
| 电池输出 45 -- 54.6V | 3 | 负 |
| | 4 | 负 |
| | 5 | NC |
| | 6 | NC |
